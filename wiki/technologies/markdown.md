# Markdown

## Technology Overview
Markdown is a lightweight markup language for writing structured text that can be rendered to HTML and other formats.

## Design Goals
- Human-readable plain text
- Fast authoring for technical writing
- Portable rendering across tools

## Architecture and Core Mechanisms
Markdown uses plain-text syntax for headings, lists, links, emphasis, and code blocks. Parsers convert this syntax into structured output.

## Usage
Common uses include documentation, README files, knowledge bases, and note systems such as Obsidian.

## Comparison with Alternatives
Compared with HTML, Markdown is faster to write but less expressive for advanced layout and styling.

## Common Questions
Different implementations may vary (for example CommonMark vs GitHub Flavored Markdown).

## Practical Guidance
Use a consistent flavor and linting rules in collaborative repositories.
