# Open Source

## Definition
Open source is a software distribution model where source code is available under licenses that allow use, study, modification, and redistribution.

## Scope and Boundaries
Open source is not identical to source-available licensing. Open Source Initiative criteria define whether a license qualifies as open source.

## Core Principles
- Transparent source code access
- Rights to modify and redistribute
- Non-discriminatory licensing terms

## Typical Cases
- Linux distributions
- PostgreSQL and other OSS databases
- Infrastructure projects such as Kubernetes

## Misconceptions and Clarifications
- Open source does not guarantee zero cost for operations.
- Open source does not guarantee automatic security.
